# 3D Character Controller

This is a simple 3D character controller made using **HTML, CSS, JavaScript, and Three.js**.

### Features
- Loads a 3D character model 
- Basic character animation
- Camera rotation using mouse
- Movement buttons:
  - Forward
  - Backward
  - Left
  - Right
  - Jump
  - Crouch
- Simple UI with buttons

### Folder Contents
- `index.html` ---> Main webpage
- `style.css` ---> Styles for UI
- `script.js` ---> Three.js logic and character movement

### How to Run
1. Download the project or clone the repository.
2. Open the folder in **VS Code**.
3. Install the **Live Server** extension.
4. Right-click `index.html` --> **Open with Live Server**.
5. The project will open in your browser.

### Notes
- Needs a local server to load 3D models.
- Works on desktop browsers (Chrome, Edge, Firefox).
